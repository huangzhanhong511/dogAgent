# 自制雪纳瑞犬专用狗饭指南

## 适用对象
- **适合**: 迷你雪纳瑞（Miniature Schnauzer）、标准雪纳瑞（Standard Schnauzer）、巨型雪纳瑞（Giant Schnauzer）
- **不适合**: 对特定食材过敏的雪纳瑞犬
- **体重范围**: 5-35公斤
- **喂食频率**: 每日2-3次

## 营养分析
| 营养成分 | 含量（每100g） | 占比 |
|---------|---------------|------|
| 蛋白质 | 22g | 22% |
| 脂肪 | 10g | 10% |
| 碳水化合物 | 50g | 50% |
| 纤维 | 3g | 3% |
| 水分 | 12g | 12% |
| 热量(kcal) | 350 | 100% |

## 食材清单
| 食材 | 用量 | 说明 |
|------|------|------|
| 鸡胸肉 | 200g | 去皮去骨，富含优质蛋白质 |
| 糙米 | 100g | 提供碳水化合物和纤维 |
| 胡萝卜 | 50g | 切丁，提供维生素A |
| 西兰花 | 50g | 切小块，富含维生素C和纤维 |
| 橄榄油 | 1汤匙 | 提供健康脂肪 |
| 鸡蛋 | 1个 | 提供额外蛋白质和脂肪 |
| 钙粉 | 1茶匙 | 促进骨骼健康 |

## 制作步骤
1. **准备食材**: 将鸡胸肉切成小块，胡萝卜切丁，西兰花切小块。
2. **烹饪**: 
   - 将鸡胸肉放入锅中，加水煮熟，捞出备用。
   - 在同一锅中加入糙米和适量水，煮至米饭熟透。
   - 在米饭快熟时，加入胡萝卜和西兰花，继续煮至蔬菜变软。
   - 在煮好的米饭和蔬菜中加入煮熟的鸡肉块，搅拌均匀。
   - 打入鸡蛋，快速搅拌均匀，确保鸡蛋完全熟透。
   - 最后加入橄榄油和钙粉，搅拌均匀。
3. **冷却**: 将混合物放置一旁，待其冷却至室温。
4. **分装**: 将冷却后的狗饭分装成每日所需的份量。

## 保存方法
- **冷藏**: 可保存3天
- **冷冻**: 可保存4周
- **解冻方法**: 将冷冻的狗饭放入冰箱冷藏室解冻一夜，或使用微波炉低温解冻。

## 注意事项
- 雪纳瑞犬易患胰腺炎（Pancreatitis），因此脂肪含量需控制在适当范围。
- 确保所有食材完全煮熟，以避免细菌感染。
- 定期监测雪纳瑞犬的体重和健康状况，调整食谱以满足其营养需求。

## 营养补充建议
| 补充剂 | 用量 | 用途 |

| Omega-3鱼油 | 1胶囊 | 促进皮肤和被毛健康 |

## 变体/替代
- 如果对鸡肉过敏，可替换为瘦牛肉或火鸡肉。
- 糙米可替换为藜麦（Quinoa）以增加蛋白质含量。
